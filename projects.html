SkipList

Technologies: Java

As a semester project, I was able to create a generic skip list data structure that worked well with integers, strings, and floating point values. The main goal of the project was to learn how to use different Java interfaces and libraries to implement something unique. The skip list included O(logn) operations such as: add, search, and delete; As well as O(n) operations such as: rebalance and hashCode. It required strong expierence with inheritance, incapuslation, and object-oriented programming.

2-4 Tree Data Structure

Technologies: Java

A 2-4 tree structure is a tree data stucture where each node has between two and four links to other nodes and between one and three values stored in each node. The main goal of this project was to be able to implement an advance data structure that maintained balance between both left and right side of the tree. Adding, searching, and deleting for this 2-4 tree were all O(logn) operation. This was by far one of the hardest challenges and one where I learned alot.
